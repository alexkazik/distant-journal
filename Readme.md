## Translation

If you want to help with the translation of the website, either update a `*.csv` file in [msg](msg) or copy the
english one and place your language in it.
And then either create a pull request, an issue or contact me (see below).

For translations of the game (quests) please contact me (via [email](mailto:distant-journal+7510@tx0.eu)
or [BGG](https://boardgamegeek.com/geekmail/compose?touser=txnull)).
